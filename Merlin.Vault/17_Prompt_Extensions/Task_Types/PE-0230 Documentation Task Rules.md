# PE-0230 Documentation Task Rules

## Rules

1. Do not change runtime code.
2. Verify facts against source when documenting code.
3. Avoid generic placeholders.
4. Update indexes and backlinks.
5. Add changelog entry for meaningful vault structure changes.
