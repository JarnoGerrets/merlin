# PE-0200 Bugfix Task Rules

## Rules

1. Reproduce or identify the failing path before changing code.
2. Keep fix narrow.
3. Add regression test where practical.
4. Add/update bug note in `09_Bugs`.
5. Create agent run report.
6. Update code atlas if behavior or ownership changed.
