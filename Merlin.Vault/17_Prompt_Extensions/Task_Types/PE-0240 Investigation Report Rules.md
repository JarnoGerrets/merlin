# PE-0240 Investigation Report Rules

## Rules

1. Inspect code/logs before conclusions.
2. Separate facts, inferences, recommendations, and unknowns.
3. Create or update agent report in `07_Agent_Reports` if investigation is substantial.
4. Add bug notes if issues are found.
5. Do not implement fixes unless explicitly asked.
