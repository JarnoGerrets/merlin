# Imported Merlin.ToDo

This folder contains copied/imported material from the legacy `Merlin.ToDo` project folder.

`Merlin.ToDo` is retained temporarily as a source backup.

New durable planning should live in:
- `13_Implementation_Plans` for extensive implementation plans.
- `08_Implementation_Prompts` for short execution prompts.
- `07_Agent_Reports` for reports.
- `03_Features`, `02_Architecture`, and `11_Code_Atlas` for curated project truth.

Do not add new long-lived planning docs to `Merlin.ToDo` unless explicitly requested.
