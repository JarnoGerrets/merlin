# Current Implemented Features

- Backend command routing with direct command matchers and tool fallback.
- Live utterance gate and barge-in/interruption pipeline.
- Assistant speech playback and UI state broadcasting.
- Memory system with EF persistence, prompt compilation, retrieval, and user facts.
- Browser Workspace with WebView2 BrowserHost, navigation, scroll, zoom, search, snapshots, common actions, and safety guard.
- Active Surface Layer with Dashboard, BrowserWorkspace, and Unknown surfaces.
- Motion Control Profile Layer with Dashboard, BrowserWorkspace, and Neutral profiles.
- Vision Sidecar with tracking and calibration paths.
- Dashboard UI window system in Godot.
- Trusted registry for applications/URLs/commands.

Implemented means code and tests were found, not necessarily flawless production quality.
